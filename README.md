# 🗺️ Restaurant Location Analysis

Geographical analysis of 9,542 restaurants across 
15 countries and 140 cities using the Zomato dataset.

## 🌍 Key Findings
- **New Delhi dominates** with 57.4% of all restaurants (5,473!)
- **London** has the highest avg rating: 4.54 ⭐
- **Colombo** is the most expensive city: avg cost 2,375 for two
- **Price range** correlates most with rating (0.438)
- Dataset spans from Hawaii (-157°) to New Zealand (174°)!

## 📊 Analysis Performed
- Interactive map with 2,000 restaurant markers + heatmap
- City-level statistics (count, rating, cost, votes)
- Correlation analysis between features and ratings
- Online delivery distribution by city
- Global geographic spread visualization

## 🛠️ Tech Stack
Python | Pandas | Folium | Matplotlib | Seaborn | HTML | Chart.js
